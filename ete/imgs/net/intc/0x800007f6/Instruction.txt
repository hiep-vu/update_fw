Steps to flash LAN EEPROM:

1. If you have 3rd party LAN controller, please remove to prevent flashing on wrong target.

2. Start up Linux64.

3. If install, nvmupdate64e, and MTG.sh are NOT executeable, execute "chmod +x install nvmupdate64e MTG.sh".

4. Execute "./install".

5. Execute "./MTG.sh".

6. AC recycle.
